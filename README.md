  ## Description
  ### A single page application that meets PWA criteria as a in browser text editor. Features data persistence techniques and works offline. This can also be installed on the user's local system

  ## Installation Instructions
  ### Clone the repo. Open integrated terminal and run 'npm run build' the 'npm run start' to initialize. Open application with your browser's localhost
